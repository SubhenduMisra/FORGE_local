# FORGE v4 — Multi-GPU AI Development Studio

<p align="center">
  <strong>Federated Orchestration for Rapid Generative Engineering</strong><br>
  MacBook M4 Max + DGX Spark + RTX 6000 Pro + Claude Cloud
</p>

## One-Command Install

```bash
curl -fsSL https://raw.githubusercontent.com/SubhenduMisra/FORGE_local/main/install.sh | bash
```

## Features

| Node | Hardware | Role |
|------|----------|------|
| **MacBook M4 Max** | 128GB unified | Quick edits, autocomplete, embeddings |
| **DGX Spark** | GB10 128GB | Heavy code gen, STT, TTS, music, vision |
| **RTX 6000 Pro** | GB202 96GB | Image/video gen, training, deep reasoning |
| **Claude Cloud** | API | Architecture, code review, security audit |

## Skills

- **Code Review**: SOLID principles, best practices, P0-P3 severity
- **Security Audit**: OWASP Top 10, secrets detection, CVE scanning

## Commands

```bash
forge status          # Show all nodes
forge code            # Launch Claude Code with routing
forge review <path>   # Run code review
forge security <path> # Run security audit
forge spark ssh       # SSH into DGX Spark
```

## Dashboard

Press `?` for help, `/` for command palette, `1-8` for tabs.
