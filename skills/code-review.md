# Code Review Skill

## Usage
```bash
forge review <path>
# or in Claude Code: /review
```

## Checks
- SOLID principles (SRP, OCP, LSP, ISP, DIP)
- Dead code detection
- Error handling
- Performance issues
- Documentation coverage

## Severity
- P0: Critical (security, data loss)
- P1: High (bugs, major issues)
- P2: Medium (code smells)
- P3: Low (style, minor)
